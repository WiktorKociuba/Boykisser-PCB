# Boykisser-PCB

A simple PCB, in shape of a boykisser, with 2 LEDs and a photoresistor that controls the circuit.

### BOM

- 1x CR2032 battery cell
- 2x 2mm LED
- 1x 2N3904 transistor
- 1x 47k resistor
- 4x 220 resistor
- 1x GL5528 photoresistor
- 1x SPST THT switch
